# Custom Controllers
These are based on the GP2040CE advanced breakout, and use the GP2040CE firmware. To program the boards, follow the GP2040CE installation instructions. The controller is connected to the motherboard over a standard usb interface, and should act as a standard usb controller.

Note for ordering: The impedance requirements for the pcb should be fairly loose, but to be safe I recommend using the JLC 7628 stackup when ordering the boards.
